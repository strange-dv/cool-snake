# Cool Snake

Terminal snake game built with Rust and Ratatui.

![Demo](demo/demo.gif)

## Build

```
cargo build --release
```

## Run

```
cargo run --release
```

## Controls

- `W` / `K` / `Up` - Move up
- `S` / `J` / `Down` - Move down
- `A` / `H` / `Left` - Move left
- `D` / `L` / `Right` - Move right
- `Space` / `Enter` - Pause / Resume / Restart
- `Q` / `Esc` - Quit
